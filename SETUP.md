# Инструкция по установке и запуску

## Быстрый старт

1. Откройте терминал в папке проекта `cda-medical-card`

2. Установите зависимости:
```bash
npm install
```

3. Запустите проект в режиме разработки:
```bash
npm run dev
```

4. Откройте браузер по адресу: http://localhost:3000

## Возможные проблемы

### Проблема с картой России
Если карта не загружается, это может быть связано с:
- Отсутствием подключения к интернету (карта загружается с внешнего источника)
- Блокировкой CORS в браузере

**Решение**: Используйте локальный файл с топологией России или настройте прокси в `vite.config.js`

### Проблемы с 3D кубом
Если 3D куб не отображается:
- Убедитесь, что браузер поддерживает WebGL
- Проверьте консоль браузера на наличие ошибок

### Проблемы с зависимостями
Если возникают ошибки при установке:
```bash
# Очистите кэш и переустановите
rm -rf node_modules package-lock.json
npm install
```

## Структура компонентов

- **RussiaMap** - Интерактивная карта России
- **WindRose** - Розы ветров со статусами
- **Hypercube3D** - 3D интерактивный куб
- **CashFlowChart** - Графики Cash Flow и добычи
- **LifecycleChart** - График жизненного цикла

## Настройка

Все параметры можно изменять в соответствующих компонентах:
- Данные регионов: `src/components/RussiaMap.jsx`
- Статусы объектов: `src/components/WindRose.jsx`
- Параметры куба: `src/components/Hypercube3D.jsx`
- Данные графиков: `src/components/CashFlowChart.jsx` и `LifecycleChart.jsx`
