# Презентация руководству: Конфигуратор систем и n8n

## Тезисы для демо

1. **Конфигуратор систем** реализован на базе концепции workflow-редактора (как в n8n): единая модель сценария, три синхронных представления — **Код** (Mermaid), **Схема** (визуализация) и **Планирование** (BPM-доска).

2. При изменении данных на вкладке **Планирование** (этапы, карточки, системы) автоматически обновляются **Код** и **Схема** в Конфигураторе. Редактирование кода в Конфигураторе отражается на схеме.

3. Использованы открытые технологии (Mermaid, React). Внутреннее использование в компании соответствует лицензированию. При необходимости интеграции с движком n8n возможна доработка (мост между нашей моделью и workflow n8n).
